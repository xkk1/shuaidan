该项目是一个简单的排行榜页面，包含 HTML、CSS 和 JavaScript 文件，用于展示某种排名信息。

### 文件说明

- **rank.html**: 页面的结构文件，定义了页面的基本布局和元素。
- **rank.css**: 样式文件，控制页面的外观和样式。
- **rank.js**: 脚本文件，用于动态渲染排行榜数据。

### 功能说明

- `renderRankTable(data)`: 根据传入的数据渲染排行榜表格。
- `getRankData()`: 异步获取排行榜数据。
- `refreshRank()`: 刷新排行榜数据。

### 使用方法

1. 打开 `rank.html` 文件即可在浏览器中查看排行榜页面。
2. 确保 `rank.css` 和 `rank.js` 文件在正确的路径下，以便正确加载样式和脚本。

该项目适用于需要展示简单排行榜信息的场景，如游戏排行榜、竞赛排名等。
