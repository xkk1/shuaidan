This project is a simple ranking page containing HTML, CSS, and JavaScript files for displaying ranking information.

### File Description

- **rank.html**: The structural file defining the basic layout and elements of the page.
- **rank.css**: The stylesheet file controlling the appearance and styling of the page.
- **rank.js**: The script file used to dynamically render ranking data.

### Features

- `renderRankTable(data)`: Renders the ranking table based on the provided data.
- `getRankData()`: Asynchronously fetches ranking data.
- `refreshRank()`: Refreshes the ranking data.

### Usage

1. Open the `rank.html` file in a browser to view the ranking page.
2. Ensure that the `rank.css` and `rank.js` files are located in the correct paths so that styles and scripts load properly.

This project is suitable for scenarios requiring simple ranking displays, such as game leaderboards or competition rankings.
